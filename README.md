### K-Nearest Neighbors (KNN) Classifier
`knn-classifier/`

`decision-tree-cart/`
- **Part 1:** Decision tree on Bornova housing dataset — confusion matrix, feature importance, tree visualization

### Decision Tree Classification Assignment
`decision-tree-assignment/`
- Full decision tree pipeline: data preprocessing, training, evaluation
- Visualization of learned tree structure

### MLP & K-Means Clustering
`mlp-and-kmeans-clustering/`
- **Part A:** Multi-Layer Perceptron (MLP) classifier
- **Part B:** K-Means clustering (k=3) with 2D cluster visualizations
  - Head circumference vs Height
  - Head circumference vs Weight
  - Height vs Weight
