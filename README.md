## Contents

### Week 1 — K-Nearest Neighbors (KNN)
`week1-knn/`
- KNN classifier implementation from scratch
- Rule analysis on gender/height/weight dataset
- Visualization of classification results

### Week 2 — Decision Trees & CART
`week2-decision-tree/`
- **Part 1:** Decision tree on Bornova housing dataset — confusion matrix, feature importance, tree visualization
- **Part 2:** CART implementation — tree pruning, performance analysis

### Assignment — Decision Tree Classification
`assignment-decision-tree/`
- Full decision tree pipeline: data preprocessing, training, evaluation
- Visualization of learned tree structure

### Midterm — MLP & K-Means Clustering
`midterm-mlp-kmeans/`
- **Part A:** Multi-Layer Perceptron (MLP) classifier
- **Part B:** K-Means clustering (k=3) with 2D cluster visualizations
  - Head circumference vs Height
  - Head circumference vs Weight
  - Height vs Weight
